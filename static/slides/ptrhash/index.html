Problem statement

Given a set of \(n\) keys \(K\subseteq \mathbb K\), build a function \(h\) satisfying

  • \(h(K) = \{0, \dots, n-1\}\), i.e., \(h\) is minimal & perfect.


Goals, in this order:

  • \(h\) is fast to evaluate,
  • \(h\) is fast to construct,
  • \(h\) takes little memory to store.
